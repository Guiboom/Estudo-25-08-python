# Estudo-25-08-python

Repositório contendo exercícios e exemplos desenvolvidos durante meus estudos iniciais de Python.

## Objetivo

Praticar lógica de programação e aprender os conceitos fundamentais da linguagem Python por meio de exercícios simples.

## Conteúdo

Os arquivos abordam temas como:

- Entrada e saída de dados
- Operações matemáticas
- Conversão de unidades
- Condicionais (`if` e `else`)
- Cálculo de salários
- Descontos em produtos
- Controle de velocidade
- Tempo de viagem
- Exercícios diversos de lógica

## Tecnologias

- Python

## Observação

Este repositório foi criado durante minha fase inicial de aprendizado. Os códigos foram mantidos como registro da minha evolução e podem não refletir minhas práticas atuais de programação.
